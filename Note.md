1.3 - typescript configuration in the ts.config.json

- `"rootDir": "./except-folder(name or path)/"` Specify the root folder within your source files
- `"outDir": "./output-folder(name or path)/"` Specify an output folder for all emitted files.
- `"target": "es2016"` Set the JavaScript language version for emitted JavaScript and include compatible library declarations.
